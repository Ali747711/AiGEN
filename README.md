A beautiful, fast, and fully typed **AI chat web app** powered by **Google Gemini 1.5 Flash**, built with **Express.js + TypeScript + EJS**.

Fully styled responses with **bold, lists, code blocks, tables, syntax highlighting** — just like ChatGPT!

Live demo → https://your-username.github.io/gemini-ai-chat (or deploy link later)

## Features

- Real-time chat with Google Gemini AI
- Markdown → Beautiful HTML rendering (bold, lists, code, tables)
- Syntax highlighting for code blocks
- ChatGPT-style UI with user/AI bubbles
- Full TypeScript support
- Clean, production-ready code
- EJS templating (no React needed!)
